# Projeto UNES – Versão Nova

Este projeto é uma recriação moderna do site da **UNES**, desenvolvido como exercício de HTML, CSS e um pequeno trecho de JavaScript.  
A proposta foi aplicar o conceito **Mobile First**, criando uma interface otimizada para telas pequenas e ampliando para desktop.

---

## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
  - Mobile First  
  - Flexbox  
  - Transitions  
  - Media Queries  
- **JavaScript**
  - Menu hambúrguer responsivo

---

## O que foi desenvolvido

### Mobile First  
O layout foi construído inicialmente para dispositivos móveis, garantindo boa usabilidade em telas pequenas.

### Menu Hambúrguer  
Implementado com JavaScript para:

- Exibir/ocultar o menu em telas menores  
- Manter o layout limpo e funcional  
- Melhorar a navegação mobile

### Estilização com CSS  
Inclui:

- Transições suaves (`transition`)
- Organização com Flexbox
- Responsividade pelo uso estratégico de Media Queries

### Outras Páginas  
As demais telas foram criadas com foco em prática de:

- Estruturas de formulário  
- Organização de conteúdo  
- Navegação entre páginas (`href`)
